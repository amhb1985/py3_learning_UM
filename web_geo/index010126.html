HTML – Allgemeiner Überblick

HTML (HyperText Markup Language) ist die grundlegende Sprache zur Strukturierung von Webseiten.

1. Semantische Tags

Diese Elemente geben dem Inhalt eine inhaltliche Bedeutung.

  • <header>
  • <nav>
  • <article>
  • <section>
  • <footer>
  • <aside>

2. Hierarchische Tags

  • <head>
  • <body>
  • <main>
  • <h1> – <h6>
  • <ul> / <li>

3. Verknüpfungen

  • <link>
  • <a>
  • <img>

4. Textauszeichnung

  • <p>
  • <span>
  • <sub>
  • <sup>
  • <br>

CSS – Grundlagen

CSS (Cascading Style Sheets) beschreibt das Aussehen von HTML-Elementen.

CSS-Regel

Eine CSS-Regel besteht aus:

  1. Selektor
  2. Eigenschaft
  3. Wert
